# Klokking Management System (2026 Bèta)
**[Switch to Dutch Version](./README_NL.md)**

Klokking is a competition registration and management system specifically developed for canoe sprint and canoe marathon events.
The platform supports online registrations, competition management, and the online publication of results, providing organizers with an efficient and reliable way to manage events from start to finish.

The Klokking Management System aims to simplify event administration while offering participants and spectators clear and accessible competition information.

>This project is currently in **bèta**, meaning functionality may change as development progresses.
